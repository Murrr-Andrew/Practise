To make eslint work:

Needs to create file .eslintrc.json with content:

{
    "extends": "react-app"
}

Also, in vite.config.js needs to add:

import eslint from "vite-plugin-eslint";

and add to array => eslint()

should like this:

export default defineConfig({
  plugins: [react(), eslint()],
});
